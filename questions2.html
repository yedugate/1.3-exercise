Problem 1. Pairing students

↑ Top

Question

A teacher wants to arrange students in pairs for a classroom activity.
Show that the number of students is either of the form:

\[2q\]

or

\[2q+1\]

Answer / Solution

If the number of students is divided by \(2\), the remainder is either \(0\) or \(1\).
So the number of students is either:

\[2q \quad \text{or} \quad 2q+1\]

Problem 2. Chocolates shared among 3 children

↑ Top

Question

A box has some chocolates. When shared equally among \(3\) children, some chocolates may remain.
Show that the number of chocolates is of the form:

\[3q,\quad 3q+1,\quad 3q+2\]

Answer / Solution

By Euclid’s Division Lemma:

\[a = 3q+r,\quad 0 \le r < 3\]

So,

\[r=0,1,2\]

Hence,

\[a = 3q,\quad 3q+1,\quad 3q+2\]

Problem 12. Square garden with odd side length

↑ Top

Question

A square garden has an odd number of metres as its side length.
Show that its area is of the form:

\[4q+1\]

Answer / Solution

Let the side length be an odd integer.

\[x=2m+1\]

Area:

\[x^2=(2m+1)^2\]
\[x^2=4m^2+4m+1\]
\[x^2=4(m^2+m)+1\]

Let:

\[q=m^2+m\]

So,

\[x^2=4q+1\]

Problem 13. Square tiles on a floor

↑ Top

Question

A square floor is made using \(n^2\) tiles.
Show that if \(n\) is any positive integer, then \(n^2\) is of the form:

\[3m\]

or

\[3m+1\]

Answer / Solution

Any positive integer \(n\) is:

\[3q,\quad 3q+1,\quad 3q+2\]

Squaring these forms gives:

\[(3q)^2=9q^2=3m\]
\[(3q+1)^2=9q^2+6q+1=3m+1\]
\[(3q+2)^2=9q^2+12q+4=3m+1\]

Hence,

\[n^2=3m \quad \text{or} \quad 3m+1\]

Problem 18. Cubical boxes stacked in layers

↑ Top

Question

A shopkeeper stacks cubical boxes. If the number of boxes along one edge is \(x\), total boxes are \(x^3\).
Show that \(x^3\) is of the form:

\[9m,\quad 9m+1,\quad 9m+8\]

Answer / Solution

Any positive integer \(x\) is:

\[3q,\quad 3q+1,\quad 3q+2\]

Cubing:

\[(3q)^3=27q^3=9m\]
\[(3q+1)^3=27q^3+27q^2+9q+1=9m+1\]
\[(3q+2)^3=27q^3+54q^2+36q+8=9m+8\]

Hence,

\[x^3=9m,\quad 9m+1,\quad 9m+8\]
Intermediate

Problem 21. Library shelves and square numbers

↑ Top

Question

A librarian arranges books in a square pattern. One student says the total number of books is \(3m+2\).
Show that this is impossible if the arrangement is a perfect square.

Answer / Solution

Let the number of books on one side be \(a\).
Total books:

\[a^2\]

Now,

\[a=3q,\quad 3q+1,\quad 3q+2\]

Then:

\[(3q)^2=9q^2=3m\]
\[(3q+1)^2=9q^2+6q+1=3m+1\]
\[(3q+2)^2=9q^2+12q+4=3m+1\]

So:

\[a^2=3m \quad \text{or} \quad 3m+1\]

It can never be:

\[3m+2\]

Hence the student is wrong.

Problem 22. Stadium block seating

↑ Top

Question

A stadium block has seats arranged in a square. The total number of seats is an odd square.
Show that the total number of seats is of the form:

\[8q+1\]

Answer / Solution

Let the side length be odd:

\[x=2m+1\]

Then:

\[x^2=4m^2+4m+1\]
\[x^2=4m(m+1)+1\]

Since \(m\) and \(m+1\) are consecutive integers, one of them is even.

So:

\[m(m+1)=2q\]

Therefore:

\[x^2=4(2q)+1\]
\[x^2=8q+1\]

Hence, the square of an odd integer is of the form:

\[8q+1\]
